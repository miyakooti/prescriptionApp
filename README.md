# QR表示デモアプリ

### 動作環境
- Xcode 14.3.1
- cocoapods 1.11.3

### ビルド方法
```
$ pod install
```

### アピールてきなもの
- .gitignoreされてる
- プレゼンテーションロジックはview側に可能な限り分離した
- extensionの記法でプログラムを分割した
- アプリ共通定数はConstants.swiftを利用
- 画面遷移をstoryboardで行わずコードで行うことで、ファイルを分割
    - 保守性を担保、コンフリクトの防止。
- if文ではなくguard文で変数存在判定をすることで、ネストをなるべく浅く
- 適切なアクセス修飾子    
- 洗練されたUI/UX
